# asteroid
Classic Asteroid Game coded with SDL2 in C
* Remember to install GCC and SDL2 libraries to compile.
* MIT LICENSE added

TO COMPILE and RUN:  

    * Download or clone repository.
    * Type "bash build.sh" or "make" and "./asteroid" to run.
    *  ** New ** Windows 32-bit release!
